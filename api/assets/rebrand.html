Compliant Cleaning Supplies

SDS Rebrand Tool

How to use this tool

1
Upload your files

Drag and drop one or more SDS documents onto the upload area, or click it to browse. Supports .docx and .pdf files. You can upload multiple files at once.

2
Click "Rebrand files"

The tool processes each file and replaces all supplier branding with CCS branding — logo, supplier name, and contact details — automatically.

3
Download rebranded files

Once processing is complete, download each file individually or use Download all to save everything at once. Files are named originalname_ccs_branded.

What gets replaced

DOCX
  • Supplier logo → CCS logo
  • Supplier name & address block
  • SDS issue / revision date
  • All body text references to the old supplier
  • Emergency contact phone numbers
PDF
  • Supplier name text (all pages)
  • First header logo image → CCS logo
  • Hyperlink URLs
  • SDS issue / revision date
  • Note: font styling may differ slightly from original
Only upload SDS files that CCS has the rights to rebrand. Do not upload third-party safety documents unless you have written authorisation to modify them.
